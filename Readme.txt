Random Password Generator
A simple Python script to generate random passwords using the random module.

Overview
This script allows users to generate random passwords of a specified length. It utilizes a character set (pass_data) that includes lowercase letters, digits, and various special characters. The random.sample() function ensures that each character in the password is unique.

Usage
Clone the repository:

bash

git clone https://github.com/your-username/random-password-generator.git
Navigate to the project directory:

bash
cd random-password-generator
Run the script:

bash

python password_generator.py
Enter the desired length of the password when prompted.

The generated password will be displayed on the console.

Example
bash
the length of your password: 12
